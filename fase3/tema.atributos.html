Atributos de las etiquetas HTML



        Los atributos de las etiquetas HTML son palabras especiales que se usan dentro de la etiqueta(en la apertura) para controlar el comportamiento que 
        tendrá la etiqueta a la cual fue asignada esta propiedad.

        ejemplo: () = <>

            (a href="")(a) - en este caso, el atributo a la etiqueta "a" es "href" en cuando al comportamiento, el valor asignado entre las comillas
                                    será la direccion a la que enviará al usuario el enlace
